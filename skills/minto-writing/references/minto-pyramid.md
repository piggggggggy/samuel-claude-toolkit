# Pyramid Structure

## Basic Form

```
           [Answer / Core Message]
          /          |          \
   [Key Line 1] [Key Line 2] [Key Line 3]
      / \            |           / \
  [detail][detail] [detail] [detail][detail]
```

## Pyramid Rules

1. **Vertical rule**: Lower levels must explain or support the level above.
2. **Horizontal rule**: Items at the same level must be the same kind of thing. (MECE)
3. **Apex rule**: There can only be one message at the top. Two or more means the piece is split.

## Key Lines Logic

**Inductive**: Multiple cases/evidence → Answer
- Key Lines are independent evidence that each support the Answer
- "First... second... third..." structure
- "These three reasons support the Answer"

**Deductive**: Premise → reason → Answer
- Key Line 1 is the premise, Key Line 2 is the reason, Answer is the conclusion
- "A is true. B is true. Therefore C."
- Each Key Line becomes the premise for the next

---

## Three Ordering Types for Groups

Use when stuck on how to arrange Key Lines.

**Time order**: The flow of actions or steps
- "First → then → finally"
- MECE test: Are any steps missing?
- Best for: process explanations, action plans

**Structural order**: Spatial or organizational breakdown of a whole
- Org charts, system components, geographic breakdowns
- Easiest MECE check — parts should add up to the whole
- Best for: situation analysis, component descriptions

**Importance order**: By size, impact, or priority
- "Most important first"
- MECE test: Can you explain why this outranks that?
- Best for: recommendations, prioritization proposals

---

## MECE Principle

**Mutually Exclusive, Collectively Exhaustive** — no overlaps, no gaps.

### Patterns Where MECE Breaks Down

**Overlap (ME violation)**
- "Reduce marketing costs" and "reduce advertising spend" at the same level
- One item is a subset of another

**Gap (CE violation)**
- Three solutions proposed but one important option is missing
- The frame is too narrow to cover the full space

**Mixed criteria**
- Geographic grouping and functional grouping at the same level
- Inconsistent basis for classification

### MECE Check Question
"Why do these belong at the same level?" — If it can't be explained, MECE is broken.
