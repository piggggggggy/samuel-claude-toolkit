# Logic Verification Tools

Use when the pyramid structure is in place but something still feels off, or when inductive/deductive logic is wobbling.

---

## Vertical Logic Check

Does each lower-level item answer "why?" for the level above?

**How to test**: Read the higher-level message and ask "why?" — do the items below answer that question?

**Common error**:
- Above: "We need to do A"
- Below: "Here are three ways to do A" → **vertical logic violation**
- The lower level should not answer "how." It should answer "why A."

Mixing "why?" answers and "how?" answers at the same level collapses the pyramid. They need to be on separate levels.

---

## Horizontal Logic Check

Are items at the same level genuinely the same kind of thing?

### Inductive Horizontal Logic

**Summation test**: When Key Lines are combined, does the Answer emerge?

> Key Line 1 + Key Line 2 + Key Line 3 → "Therefore [Answer]"

If it doesn't feel natural, the grouping is wrong.

**Same-kind test**: Are the items truly in the same category?

Wrong:
> "Cut costs," "grow revenue," "boost team morale" → Adding these up doesn't produce a single Answer

Right:
> "Cut headcount costs," "cut operations costs," "cut marketing costs" → Adding these up produces "overall cost reduction"

### Deductive Horizontal Logic

**Predicate test**: Does the second item speak to the **predicate** of the first?

Correct deduction:
> "The market is growing (premise). We have the capability to capture share from that growth (statement about the predicate). Therefore we should enter now."

Incorrect deduction (becomes parallel listing):
> "The market is growing. There are many competitors. Therefore we have an opportunity."
> → The two statements are independent. The first is not a premise for the second.

## Mini-Pyramid at the Paragraph Level

Use when structure is solid but individual paragraphs still feel scattered.

Each paragraph follows pyramid structure:
- **First sentence of the paragraph** = the Answer for that paragraph
- **Remaining sentences** = evidence or explanation supporting that Answer

**How to check**: Does reading only the first sentence of the paragraph convey the paragraph's core point? Do the remaining sentences explain or support the first — or do they introduce a new topic?

If a new topic appears → the paragraph needs to be split.
